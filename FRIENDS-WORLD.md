# עולם החברים — תוכנית עבודה 🌟

קובץ העבודה המשותף שלנו. החזון: **כל האפליקציה היא "עולם החברים"** — דמויות חמודות עם אישיות, שמככבות בכל משחק, מתחברות, עושות חישובים, ומלוות את אסף בלמידה ובכיף.

> עדכון אחרון: 30/05/2026

---

## 1. החזון

- מסך הבית הוא עולם של **חברים** — לא אוסף משחקים מקריים.
- לכל חבר יש **צבע קבוע, שם, וקול/קריאת שמחה**.
- החברים **חיים**: זזים קצת, ממצמצים, קופצים ומגיבים כשנוגעים בהם.
- החברים **מתחברים** זה לזה (לחיבור, ספירה, מחשבון).
- כל משחק אחר באפליקציה הופך בהדרגה למשחק בעולם החברים.

## 2. עקרונות (לא משתנים)

- **פשוט להפעלה** — מסך אחד, מטרה אחת, מעט כפתורים גדולים.
- **בלי זמן ובלי לחץ** — אף פעם. גם משחקי "מירוץ" יתקדמו לפי הצלחה, לא לפי שעון.
- **מבוסס תמונות + קול בעברית** (אסף לא קורא). אפשר לכבות קול בהגדרות.
- **דמויות מקוריות לגמרי** — לא חיקוי של אף סדרה. זהות משלנו.
- **רגוע וידידותי** — בלי הבהובים מפתיעים, בלי צליל "טעות" מעניש. תמיכה ב-prefers-reduced-motion.
- עברית, RTL, מובייל-פירסט (אבל גם יפה באייפד).

## 3. הקאסט — החברים 🎨

רוֹסטר קבוע של 10 חברים. כל כמות מוצגת ע"י החברים הראשונים (חבר 1 תמיד אותו צבע/שם) — כך אסף מזהה אותם.

רוֹסטר קבוע (ב-`src/friends.ts`). **כל חבר בנוי ממספר יחידות** — לולו = יחידה 1, טוקי = 2 יחידות, בובי = 3 ... כך הגובה והצורה מראים את המספר (עיצוב מקורי משלנו: יחידות מעוגלות, פרצוף, צבעים ושמות שלנו — לא העתקה). על כל חבר גם תג מספר קטן. זה מה שעוזר בספירה/חיבור/חיסור (האחרון שנספר = הסכום).
לדיבור יש גרסה **מנוקדת** (`say`) כדי שה־TTS יבטא נכון; השם המוצג בלי ניקוד. קל לשנות כל שם/הגייה במקום אחד.

| מס' | צבע | שם | דיבור (מנוקד) |

| 1 | אדום | לולו | לוּלוּ |
| 2 | כתום | טוקי | טוּקִי |

## 4. החלטות פתוחות (צריך אתכם)

1. **המשחקים שאינם "חברים"** — ✅ הוחלט: להמיר לעולם החברים מה שאפשר; מה שלא ניתן להמיר יועבר לתיקיית גיבוי עד החלטה (לא נמחק). זה יקרה בשלב 6.
2. **שמות החברים** — ✅ הוחלט: אני מציע (ראו טבלה), אתם מכווננים.
3. **מחשבון החברים** — ❓ אילו פעולות? (חיבור/חיסור בלבד, או גם כפל/חילוק — אסף כבר בכפל). _צריך החלטה לפני שלב 3._
4. **מאיפה מתחילים** — ✅ הוחלט: שלב 0 (אישיות + חיוּת). הושלם.

## 5. שלבי עבודה (לפי סדר מומלץ)

### שלב 0 — תשתית "החברים" 🧱 (הבסיס לכל השאר) ✅ הושלם
- [x] `friends.ts` — רוֹסטר: לכל חבר צבע + שם, ועזרי גישה.
- [x] רכיב `Friend`: נשימה עדינה (idle), מצמוץ, קפיצה + צליל + אמירת שם בנגיעה, מצב bouncing.
- [x] צליל ייחודי לכל חבר (`playFriend`) + אומר את שמו בקול.
- [x] "סופרים" ו"מוסיפים" כבר משתמשים בחברים החיים.

### שלב 1 — לשדרג את המשחקים הקיימים של החברים
- [ ] "סופרים" — חברים בעלי זהות, חיים, ספירה בקול (קיים בסיס).
- [ ] "מוסיפים" — אנימציית **התחברות**: חברים מחליקים ומתאחדים לקבוצה.

### שלב 2 — משחקים חדשים פשוטים
- [ ] **"תן כיף"** — נוגעים בחבר → קופץ ועושה קול. חושי, בלי מטרה, בלי הפסד.
- [ ] **"מי נעלם?"** — כמה חברים, אחד מתחבא, מזהים מי חסר.

### שלב 3 — מחשבון החברים 🔢 (הדגל)
- [ ] מחשבון אמיתי לילדים (כפתורים גדולים) — אסף מקיש חישוב.
- [ ] תצוגת חברים בצד שמראה את החישוב חי (כך וכך חברים).
- [ ] **רספונסיבי:** באייפד — מסך מפוצל (מחשבון + חברים בצד). בסמארטפון — חלון/פופ-אפ של החברים.

### שלב 4 — חידוני חשבון כ"מירוץ" (בלי זמן) 🏁
- [ ] חיבור / חיסור / כפל. כל תשובה נכונה מקדמת את החבר במסלול (מירוץ לפי הצלחה, לא שעון).
- [ ] רמות לפי יכולת (אסף מתקדם — כולל כפל).

### שלב 5 — "פיצוץ חברים" 🎉 (בהשראת משחקי פופ, מקורי לגמרי, בלי זמן)
- [ ] חברים מופיעים, נוגעים → קופצים/מתפוצצים בכיף. בלי שעון, בלי הפסד.
- [ ] גרסאות: לפי צבע / לפי מספר.

### שלב 6 — להפוך הכל לעולם החברים + לסדר את המסך מחדש ✅ (סבב ראשון)
- [x] מסך בית חדש = **עולם החברים**: כרטיס מוביל "החברים שלי" + מקטע "החברים" + מקטע "עוד". בלי שלב קטגוריות נפרד (לחיצה אחת למשחק).
- [x] מסך **"החברים שלי"** — היכרות עם הדמויות (לא משחק): נוגעים בכל חבר → קופץ, צליל, ואומר את שמו. השמות מוצגים גם בכתב.
- [x] המרת **זיכרון** ל"זיכרון חברים" (מתאימים זוגות חברים; החבר אומר את שמו בהתאמה).
- [x] גיבוי (לא נמחק) ב-`src/_backup/`: סדר-יום, טיטול, מוצץ, "סופרים ביחד" (חידspace ישן וכפול). מוחרג מ-tsconfig.
- [ ] **לבדוק/להמיר בהמשך:** אותיות, מילים, בועות — כרגע במקטע "עוד" (לא חברים עדיין). אפשר: בועות → "פיצוץ חברים" (שלב 5); אותיות/מילים → חבר שמארח את החידון.

## 6. חלוקת עבודה

**אני (קלוד) — בונה:**
- כל הקוד, הדמויות, האנימציות, הצלילים, המשחקים, הרספונסיביות.
- שמירה על העקרונות (פשטות, בלי זמן, נגישות).
- בנייה צעד-צעד, עם בדיקה (build) אחרי כל שלב.

**אתם — מחליטים ובודקים:**
- ההחלטות הפתוחות בסעיף 4 (שמות, מה עושים עם שאר המשחקים, פעולות מחשבון).
- **לבדוק כל שלב עם אסף** ולספר לי מה עבד / מה לחדד.
- להעביר העדפות של אסף (צבעים/דמויות/קולות שהוא אוהב).

## 7. הערות טכניות

- `friends.ts` — מקור אמת יחיד לזהות החברים (צבע/שם/קול). כל משחק מושך משם.
- רכיב `Friend` משותף לכל המשחקים (חיוּת + תגובה למגע במקום אחד).
- אנימציית "התחברות" כרכיב משותף (חיבור, מחשבון, ספירה).
- עזר רספונסיבי: `useIsWide()` — אייפד = מסך מפוצל, טלפון = פופ-אפ.
- בלי נכסים חיצוניים / מוגנים — הכל מצויר בקוד וצלילי Web Audio.

## 8. מצב נוכחי (מה קיים היום)

- מבנה קטגוריות: **חברים** / **אחר**.
- בתוך **חברים**: "סופרים", "מוסיפים" (עם רכיב `Critter` — הבסיס לחבר).
- בתוך **אחר**: אותיות, מילים, זיכרון, סופרים-ביחד, סדר יום, טיטול, מוצץ, בועות.
- תשתית: קול עברי (`speech.ts`), צלילים (`audio.ts`), הגדרות (`settings.ts`).
