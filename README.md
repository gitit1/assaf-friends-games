# 🌟 עולם החברים — assaf-friends-games

אתר משחקים חינוכי, עברי ו‑RTL, **מובייל‑פירסט**, שנבנה במיוחד עבור אסף בן ה‑4 — ילד שאוהב מספרים ומשחקי חשיבה, חזק מאוד ויזואלית, ולא קורא עדיין (אז הכול מבוסס תמונות + הקראה בעברית).

🎮 **המשחקים באוויר (Live):** https://funny-chebakia-d5b65a.netlify.app/

---

## מה זה?

כל האפליקציה היא **"עולם החברים"** — חבורה של דמויות חמודות שכל אחת מהן **היא מספר** (החבר ה‑1 הוא לולו, ה‑2 טוקי, וכן הלאה). הדמויות מצוירות לגמרי ב‑CSS (בלי תמונות), יש להן צבע, שם וקול קבועים, הן זזות ומגיבות למגע — והן מככבות בכל המשחקים. כך הלמידה (ספירה, חיבור, כפל, צבעים, צורות) קורית דרך משחק וכיף.

הקאסט כולל כרגע **30 חברים (1–30)**, והיעד להגיע עד 100. לכל חבר אביזר ייחודי משלו.

## עקרונות מנחים (לא משתנים)

- **בלי שעון ובלי לחץ** — אף משחק לא מוגבל בזמן. ההתקדמות לפי הצלחה, לא לפי שעון.
- **בלי כישלון ובלי עונש** — אין מסך "הפסדת"; טעות מקבלת דחיפה עדינה בלבד.
- **מבוסס תמונות + קול בעברית** — מתאים לילד שלא קורא; אפשר לכבות קול בהגדרות.
- **רגוע וחושי‑נעים** — בלי הבהובים מפתיעים, בלי צליל "טעות" מעניש.
- **מטרות מגע גדולות**, עברית, RTL, ויפה גם באייפד.

## טכנולוגיה

- **Vite + React 19 + TypeScript**
- **בלי נכסים חיצוניים:** כל הגרפיקה ב‑CSS, הצלילים ב‑Web Audio, וההקראה ב‑Web Speech API (he‑IL).
- שמירת מצב (הגדרות, טמגוצ'י) ב‑localStorage.
- ניווט מבוסס‑hash (רענון נשאר באותו מסך, וכפתור "אחורה" של הדפדפן עובד).
- פריסה: **Netlify**.

## הרצה מקומית

```bash
npm install
npm run dev      # שרת פיתוח
npm run build    # בנייה לפרודקשן (tsc && vite build)
npm run preview  # תצוגה מקדימה של ה-build
```

## מבנה הפרויקט

```
src/
  friends.ts              # מקור האמת לזהות החברים (צבע / שם / קול)
  components/
    FriendArt.tsx         # האמנות המשותפת (1–10 ידני, 11–30 מבוסס-נתונים)
    Friend.tsx            # עוטף את FriendArt + חיוּת ותגובה למגע
    HomeScreen.tsx        # קוביות קטגוריות
    GameShell.tsx         # מסגרת משותפת לכל משחק (כפתור בית)
    Stepper.tsx / IconButton.tsx / FullscreenButton.tsx   # רכיבים משותפים
  games/
    registry.ts           # רישום המשחקים + הקטגוריות (מוסיפים כאן משחק)
    *.tsx                 # המשחקים
  audio.ts / speech.ts / settings.ts
  app.css                 # כל העיצוב
```

**להוספת משחק חדש:** יוצרים רכיב ב‑`src/games/`, ומוסיפים רשומה ל‑`GAMES` ב‑`registry.ts` עם הקטגוריה. קטגוריה ריקה לא מוצגת.

תוכנית העבודה והחזון המלא: ראו [`FRIENDS-WORLD.md`](FRIENDS-WORLD.md).
