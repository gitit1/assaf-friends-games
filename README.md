# המשחקים של אסף 🎮

אתר משחקים פשוט וידידותי, בנוי במיוחד למובייל (אבל עובד גם בדפדפן רגיל).
המשחקים תוכננו להיות רגועים, צפויים ונטולי תסכול — **בלי טיימרים, בלי "הפסדת"**, רק חיזוקים חיוביים.
אסף לא קורא אבל חזק ויזואלית, אז הכל מבוסס תמונות + קול בעברית.

## המשחקים

### לימוד
- **🔢 מספרים וכפל** — בסגנון Numberblocks: קוביות צבעוניות. מצב "סופרים" ומצב "כפל" (קבוצות של קוביות). מתאים לרמה מתקדמת, בלי זמן.
- **🔤 חידון אותיות** — רואים תמונה, שומעים את המילה, ובוחרים באיזו אות היא מתחילה.
- **🗣️ חידון מילים** — שומעים מילה ובוחרים את התמונה הנכונה.
- **🧠 משחק זיכרון** — מוצאים זוגות תואמים. שתי רמות.
- **🐥 סופרים ביחד** — סופרים פריטים ובוחרים מספר.

### שגרה והתנהגות
- **📅 סדר היום** — רצף ויזואלי של היום (קימה → שירותים → גן → אמבטיה → שינה). מצב צפייה ("הבא") ומצב חידון ("מה עושים אחר כך?").
- **🚽 ביי ביי טיטול** — סיפור חברתי קצר + לוח כוכבים שמתמלא בכל הצלחה. ההתקדמות נשמרת.
- **👋 ביי ביי מוצץ** — אותו רעיון, נפרד לגמרי עם לוח כוכבים משלו.

### חושי
- **🫧 בועות** — מפוצצים בועות שעולות. בלי ניצחון/הפסד.

## קול בעברית 🔊

המשחקים משתמשים ב-Text-to-Speech של הדפדפן (he-IL) כדי להקריא שאלות, מילים ועידוד — כך אסף יכול לשחק לבד.
אפשר לכבות את הקול (וגם את הצלילים) בכפתור ⚙️ ההגדרות במסך הבית. אם אין קול עברי במכשיר — תופיע הודעה ואפשר להמשיך עם צלילים בלבד.

## הרצה

```bash
npm install
npm run dev      # שרת פיתוח מקומי
npm run build    # בנייה לפרודקשן (תיקיית dist)
npm run preview  # תצוגה מקדימה של הבנייה
```

## מבנה הקוד

- `src/games/registry.ts` — רשימת המשחקים. **כדי להוסיף משחק חדש**: יוצרים קומפוננטה ב-`src/games/` ומוסיפים אותה כאן. מסך הבית מתעדכן אוטומטית.
- `src/games/*.tsx` — המשחקים (כל אחד מקבל `onExit`).
- `src/games/data/` — הנתונים (אותיות, מילים, סדר יום). קל לערוך ולהוסיף.
- `src/components/GameShell.tsx` — מסגרת משותפת עם כפתור "בית".
- `src/components/RewardJourney.tsx` — לוח הכוכבים המשותף לטיטול/מוצץ.
- `src/components/NumberBlock.tsx` — קוביית מספר בסגנון Numberblocks.
- `src/components/SettingsPanel.tsx` + `src/settings.ts` — הגדרות קול/צלילים (נשמרות ב-localStorage).
- `src/speech.ts` — קול בעברית (Web Speech API).
- `src/audio.ts` — צלילים עדינים (Web Audio, בלי קבצים).
- `src/app.css` — עיצוב מובייל-פירסט עם אזורי מגע גדולים.

## עקרונות עיצוב (לילד אוטיסט)

- אזורי מגע גדולים וברורים, צבעוניות רגועה.
- בלי טיימרים ובלי לחץ; אין צליל "טעות" מעניש — רק רמז עדין.
- כל מסך כולל כפתור חזרה גדול וקבוע לבית.
- תמיכה ב-`prefers-reduced-motion`.
