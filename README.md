# 🌟 עולם החברים — assaf-friends-games

אתר משחקים חינוכי, עברי ו‑RTL, **מובייל‑פירסט**, שנבנה במיוחד עבור אסף בן ה‑4 — ילד שאוהב מספרים ומשחקי חשיבה, חזק מאוד ויזואלית, ולא קורא עדיין (אז הכול מבוסס תמונות + הקראה בעברית).

🎮 **המשחקים באוויר (Live):** https://funny-chebakia-d5b65a.netlify.app/

---

## מה זה?

כל האפליקציה היא **"עולם החברים"** — חבורה של דמויות חמודות שכל אחת מהן **היא מספר** (החבר ה‑1 הוא לולו, ה‑2 טוקי, וכן הלאה). הדמויות מצוירות לגמרי ב‑CSS (בלי תמונות), יש להן צבע, שם וקול קבועים, הן זזות ומגיבות למגע — והן מככבות בכל המשחקים. כך הלמידה (ספירה, חיבור, כפל, צבעים, צורות) קורית דרך משחק וכיף.

הקאסט כולל **100 חברים (1–100)**. לכל חבר מ‑51 ומעלה אביזר ייחודי משלו (תיק, שעון, בובה…).

## עקרונות מנחים (לא משתנים)

- **בלי שעון ובלי לחץ** — אף משחק לא מוגבל בזמן. ההתקדמות לפי הצלחה, לא לפי שעון.
- **בלי כישלון ובלי עונש** — אין מסך "הפסדת"; טעות מקבלת דחיפה עדינה בלבד. כל הצלחה חוגגת ב‑🎉 קונפטי.
- **מבוסס תמונות + קול בעברית** — מתאים לילד שלא קורא; אפשר לכבות קול בהגדרות.
- **רגוע וחושי‑נעים** — בלי הבהובים מפתיעים, בלי צליל "טעות" מעניש; מצב "תנועה רגועה" בהגדרות.
- **דו‑לשוני** — כל הטקסטים עוברים מערכת i18n: **עברית (RTL)** ו‑**אנגלית (LTR)**, מתחלף בהגדרות (הקריינות עדיין עברית).
- **מטרות מגע גדולות**, RTL נכון, ויפה ורספונסיבי בטלפון / טאבלט / מחשב.

## המשחקים

### 🔢 מספרים
קפיצות (לוח הכפל על ציר מספרים) · עשרות ואחדות (ערך‑מקום) · בונים מספר (➕➖✖️➗) · המספר החסר (a + ? = c) · גדול או קטן? · חבר חסר ברצף · מספר וכמות · מחשבון חברים · מרוץ מכוניות · אתגר חשבון

### 🧠 חשיבה
זיכרון חברים (היפוך‑קלף 3D, 4 רמות) · מי נעלם? · מיון (צבע/מספר/אות/חבר) · תבניות

### 🎉 כיף
תופסים חבר · פיצוץ חברים · פסנתר חברים (9 שירים) · החבר שלי (טמגוצ'י) · מגלגלים קובייה

### 🎨 יצירה
צובעים חבר · חיבור נקודות · ציור חופשי (עם מדבקות חברים) · צביעה לפי מספר · מציירים מספר

### 🔤 אותיות
איזה חבר? (אות עברית → החבר ששמו מתחיל בה)

### 🏀 ספורט
זריקה לסל · בעיטה לשער · הוקי אוויר · באולינג — כולם בפיזיקת "החלקה" אמיתית, בלי כישלון.

בנוסף: כרטיס **"החברים שלי"** להיכרות עם כל 100 החברים, ול**עולם של כל חבר** — שם אפשר לחבק/לתת כיף/נשיקה, לספור את הבלוקים, **לפרק את החבר לשני חברים** (c=a+b), לראות עליו **עובדה מספרית**, וליהנות מ**פרצוף חי** (עיניים שעוקבות). *(גלריית תלת‑מימד קיימת אך מופעלת רק במצב פיתוח.)*

## טכנולוגיה

- **Vite + React 19 + TypeScript**
- **בלי נכסים חיצוניים:** כל הגרפיקה ב‑CSS, הצלילים ב‑Web Audio. הקריינות העברית הוקלטה מראש (Narakeet, קול "איילת") עם נפילה ל‑Web Speech API.
- **i18n** (`src/i18n/`): מילוני he/en עם מפתחות תואמים (בדיקת build). `t('key')` / `useT()`.
- שמירת מצב (הגדרות, טמגוצ'י, מועדפים) ב‑localStorage.
- ניווט מבוסס‑hash (רענון נשאר באותו מסך, וכפתור "אחורה" של הדפדפן עובד).
- פריסה: **Netlify**.
- **סקילים לפרויקט** (`.claude/skills/`): `rtl-mirror` (מראה‑RTL) ו‑`breathing-room` (מרווחים/מובייל/פוטר).

## הרצה מקומית

```bash
npm install
npm run dev      # שרת פיתוח
npm run build    # בנייה לפרודקשן (tsc && vite build)
npm run preview  # תצוגה מקדימה של ה-build
```

## מבנה הפרויקט

```
src/
  friends.ts              # מקור האמת לזהות החברים (צבע / שם / קול)
  components/
    FriendArt.tsx         # האמנות המשותפת (1–10 ידני, 11–100 מבוסס-נתונים)
    Friend.tsx            # עוטף את FriendArt + חיוּת ותגובה למגע
    HomeScreen.tsx        # קוביות קטגוריות
    GameShell.tsx         # מסגרת משותפת לכל משחק (כפתור בית)
    Stepper.tsx / IconButton.tsx / FullscreenButton.tsx   # רכיבים משותפים
  games/
    registry.ts           # רישום המשחקים + הקטגוריות (מוסיפים כאן משחק)
    *.tsx                 # המשחקים
  audio.ts / speech.ts / voice.ts / settings.ts
  i18n/                   # he.ts + en.ts (מפתחות תואמים) + t()/useT()
  components/Confetti.tsx # חגיגת הצלחה משותפת (יורד על כל האפליקציה)
  app.css                 # כל העיצוב
```

**להוספת משחק חדש:** יוצרים רכיב ב‑`src/games/`, ומוסיפים רשומה ל‑`GAMES` ב‑`registry.ts` עם הקטגוריה. קטגוריה ריקה לא מוצגת.
