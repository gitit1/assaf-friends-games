# ⭐ הלחצן המיוחד של כל חבר → משחק

**המטרה:** לכל אחד מ‑100 החברים יש לחצן מיוחד, מותאם למה שהוא אוהב, שעושה שני דברים:
1. החבר אומר אחד מ‑**3 משפטים אקראיים** (מוקלט, בקול שלו).
2. **פותח משחק מתאים** באתר (קיים, או כזה שנבנה במיוחד).

נבנה את זה **5 חברים בכל פעם** — מתכננים מגדר + פעילות + משחק + 3 משפטים, מקליטים, ומחברים.

## כללים
- הפעילות המיוחדת **לא חופפת** ללחצנים שכבר קיימים בעולם‑החבר:
  כיף ✋ · חיבוק 🤗 · נשיקה 😘 · ספירה 🔢 · פירוק 🧩 · עובדה ✨.
- מותר ש**כמה חברים יחלקו אותה פעילות** (ויובילו לאותו משחק) — לא חייבים 100 משחקים שונים.
- אם הפעילות דורשת משחק שעוד לא קיים — **מתעדים אותו ב"משחקים לבנייה"** ובינתיים הלחצן רק אומר את המשפטים.

## הקראת עובדה לפי רמת קושי (E — לתכנון פר-חבר)

לחצן "עובדה ✨" צריך גם **להקריא** משפט (לא רק להראות מספרים), וה**משפט משתנה לפי רמת
הקושי** שבהגדרות (קל / בינוני / קשה / אלוף) — כי מה שמובן ל"אלוף" לא מובן ל"קל".

דוגמה — לולו (1):
| רמה | משפט |
|---|---|
| קל | "אני הספרה הראשונה!" · "המספר שבא אחריי הוא שתיים" |
| קשה / אלוף | "כל מספר שתכפיל בי יישאר אותו דבר" |

**הוחלט:** **4 רמות** (קל / בינוני / קשה / אלוף, לפי `difficulty` בהגדרות), **מוקלט**
בקול החבר (איכותי — לכן עובדים 1‑1 לאט). כלומר ~4 קליפי עובדה לכל חבר. נתכנן את
משפטי העובדה לכל חבר יחד עם שאר הפרטים (מגדר / לחצן+משחק / 3 משפטים מיוחדים / 4 עובדות).
