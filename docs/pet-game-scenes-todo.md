# משחק "החבר שלי" (טמגוצ'י) — צ'קליסט סצנות "משחק אמיתי"

המטרה: כל פעולה היא **סצנה מאוירת עם תנועה אמיתית** (כמו משחק 2D), הדמות בפרופורציה לרהיטים.
ענף: `tamagotchi`. הקובץ הראשי: `src/games/Tamagotchi.tsx` + הסגנונות ב‑`src/app.css`.

- [x] גודל פרופורציונלי לדמות + ממורכזת בלי חיתוך

- [x] תפאורת מטבח מלאה (ארונות, שיש, כיריים+סיר, כיור, אריחים, מקרר על הרצפה)

## ⏳ נשאר — "לפתוח את העולם"
- [ ] **משחקים — כל משחק בסצנה שלו עם תנועה:**
  - [ ] **כדור** → בחוץ עם חבר שמשחק איתו (מסירות)
  - [ ] **נדנדה** → להתנדנד בחצר
  - [ ] עוד פעילויות חוץ/פנים (טלוויזיה, מחשב, ספר, לגו — כל אחת בפינה שלה)
- [ ] **שינה** → חדר שינה עם מיטה, כיבוי אור / לילה
- [ ] **שתייה** → להראות שתייה מהמיכל המתאים (כוס/בקבוק) לכל משקה
- [ ] אופציה: **שטיפת ידיים ליד הכיור** עם ברז זורם
- [ ] ליטוש: מקרר גדול יותר, הליכה עם רגליים, עוד פרטי רקע
