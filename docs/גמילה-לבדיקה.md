### לבדוק
- [ ] **בחירת דמות:** ילד / ילדה / חבר — שלושתם נפתחים ומופיעים יפה.
- [ ] **הורדת טיטול = צעד‑צעד עם זום:** המסך **מתקרב** לדמות, **קודם המכנס** מחליק למטה ויוצא, **ואז הטיטול** — רגל‑רגל. רואים את זה קורה באמת.
- [ ] **אסלה גדולה וברורה** — מיכל, מכסה מורם, מושב, בסיס; **גדולה יותר מהכיור**.
- [ ] **בועת צורך** — "פיפי בדרך 💧" / "קקי בדרך 💩" מתחלפות בכל סבב.
- [ ] **הליכה לשירותים** — הדמות צועדת (נדנוד) ומגיעה לאסלה.
- [ ] **מורידים מכנס בזום** (גם בשירותים) → הילד יושב / הילד עומד **ומכוון** (בנים, פיפי).
- [ ] **ברור שזה פיפי** — זרם **צהוב** + שלולית צהובה בקערה (לא כחול/מים). קקי = 💩.
- [ ] **ילדה / חבר / קקי:** יושבים על האסלה — הראש נראה מעל המושב; החבר יושב על המושב.
- [ ] **מורידים מים** 🚽 → **שוטפים ידיים** 🧼 → **חגיגה** 🎉 + כוכב.
- [ ] **תאונה:** כפתור "אופס" → "לא נורא 💦" → "מנקים ומחליפים" → ממשיכים (בלי הורדת כוכב).
- [ ] **קול:** הקראה בעברית בכל שלב (מורידים מכנס/טיטול, פיפי/קקי בשירותים, כל הכבוד, לא נורא).
- [ ] **עברית + אנגלית** — טקסט וכפתורים נכונים; הסרגל העליון ממוראה נכון.
- [ ] **רוחב 390 וגם 520+** — האמבטיה, האסלה והכיור לא נחתכים; הכפתורים לא צפופים.
