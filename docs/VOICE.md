# 🎙️ קולות החברים — מצב, החלטות, ומה שנשאר

המשפטים הקבועים (היכרות כל חבר, מספרים, כפתורי "כיף/חיבוק/נשיקה" ועוד) מתנגנים
מקבצי‑קול מוקלטים מראש ב‑`public/voice/<lang>/<id>.mp3`. אם קובץ חסר — האפליקציה
נופלת לקול‑הדפדפן (`src/voice.ts` → `playClip`). המנוע: `scripts/gen-voice.mjs`.

> עודכן 06/06/2026. החלטנו: **עברית = Narakeet** (קול ישראלי טבעי). אנגלית תחכה
> לחבילת 300 הדקות (גם היא Narakeet). יוצרים **באצוות של 5 חברים** ובודקים כל פעם.

## 🎚️ פלטת הקולות (סופי)

קולות עבריים של Narakeet, לפי מגדר — בנים בקול גברי, בנות בקול נשי, ומתגוונים:

| מגדר | קולות שנבחרו | נפסלו |
|---|---|---|
| בנות 🩷 | **Ayelet · Tamar · Nurit** | Hadas, Yael (נשמעו מכניים/בלעו מילים) |

שבנות עוקבות מתחלפות Ayelet→Tamar→Nurit וחוזר חלילה, וכנ"ל בנים — **יציב לכל
חבר** (אותו חבר תמיד אותו קול, גם אם מייצרים מחדש).

## 🧠 עובדות קריטיות שלמדנו (כדי לא ליפול שוב)

1. **רק `Ayelet` ו‑`Lior` תומכים ב‑IPA.** שאר הקולות מחזירים שגיאה `400 ... does
   not support IPA phonemes`. לכן:
   - "המספר" נכתב ב**ניקוד** `הַמִּסְפָּר` (לא IPA) — כל הקולות קוראים נכון
     "ha‑mispar" ולא "mesaper". (`HAMISPAR` בסקריפט.)
   - עקיפת‑הגייה ל**שם** ב‑IPA (`NAME_IPA`) מותרת רק אם החבר משויך ל‑Ayelet/Lior.
     כרגע: לולו → `[ˈlulu]{ipa}` והיא ממילא Ayelet. ✅
2. **מילים דו‑משמעיות בלי ניקוד נבלעות/נקראות שגוי.** דוגמה: `נשחק` (לְשַׂחֵק? לְהִישָׁחֵק?).
   להעדיף צורה חד‑משמעית (`לשחק`) או לנקד.
3. **אין סגנון "נלהב" בעברית** ב‑Narakeet (רק literal/ipa/x‑sampa). ההתלהבות מושגת
   דרך **גיוון הקולות + קצב 1.0** (לא 0.9 האיטי) + סימני קריאה.
4. **הקליפ הישן הבעייתי נוצר ב‑Edge**, שהקריא מילולית את סימוני ה‑IPA (`[..]{ipa}`)
   — "פותח סוגריים מסולסלים". לכן עברנו ל‑Narakeet, ו‑`HAMISPAR` הוא ניקוד.

## ⚙️ איך מריצים (Narakeet)

המפתח שמור אצל המשתמשת כמשתנה‑סביבה ברמת User (לא בקוד, לא ב‑git). שולפים אותו
בכל ריצה דרך PowerShell:

```powershell
$env:NARAKEET_API_KEY = [Environment]::GetEnvironmentVariable('NARAKEET_API_KEY','User')
$env:VOICE_PROVIDER = 'narakeet'; $env:VOICE_LANG = 'he'
# אצווה של 5 חברים (num + intro):
$env:FROM = '6'; $env:TO = '10'; node scripts/gen-voice.mjs
```

דגלים שימושיים ב‑`gen-voice.mjs`:

| משתנה | מה עושה |
|---|---|
| `FROM` / `TO` | טווח חברים (1‑based). למשל 6..10 = num‑6..10 + intro‑5..9. |
| `ONLY=intro-2,fx-five` | מייצר רק את המזהים האלה (ריצת‑תיקון זולה). |
| `EXTRAS=1` | כולל את קליפי הכפתורים (`like-*`, `fx-*`). |
| `SAMPLE=1` | מייצר `sample-<voice>.mp3` לכל קול שבפלטה (להשמעת קולות). |
| `VOICE_SPEED` | ברירת מחדל 1.0. |
| `DRY_RUN=1` | מדפיס את הטקסט שיישלח, בלי קריאות API (בלי לבזבז דקות). |

**בדיקת יתרת דקות (חינם, לא עולה דקות):**
```powershell
$key = [Environment]::GetEnvironmentVariable('NARAKEET_API_KEY','User')
(Invoke-RestMethod 'https://api.narakeet.com/account/credits' -Headers @{'x-api-key'=$key}).creditSeconds
```
מחזיר שניות; חלקי 60 = דקות. (06/06/2026: ~3207 ש' ≈ 53.5 דק'.)

**QA מקומי:** `npm run dev` ואז פותחים `/voicetest.html` — עמוד שמנגן את הקליפים
הגולמיים ישירות (בלי מטמון, בלי האפליקציה), כולל מקטע "מחירון קולות". *(הקובץ
ודגימות `sample-*` ב‑gitignore — מקומי בלבד.)*

## 🧩 מבנה המתכון (`gen-voice.mjs`)

- `GENDER` — מפה `{ index: 'f'|'m' }`. ברירת מחדל 'm'. **מתמלא אצווה‑אצווה** לפי מה
  שההורה אומרת. משפיע גם על הקול (`voiceFor`) וגם על הטיית הפעלים בהצגה (`vg`).
- `NAME` — שמות החברים (מסונכרן עם `src/friends.ts`).
- `NAME_IPA` — עקיפת‑הגייה לשם, רק לחברים על Ayelet/Lior.
- `LIKES` / `INVITE` — מה החבר אוהב + שורת ההזמנה הסוגרת ("בואו נשיר", "אשמח לחיבוק").
- `TEMPLATES` — 5 תבניות הצגה, מוטות מגדר.
- `HAMISPAR = 'הַמִּסְפָּר'` (ניקוד).

## ⏳ מה שנשאר (לחבילת 300 הדקות)

1. **חברים 6–100** — לכל אצווה של 5: ההורה אומרת מי בן/בת → מעדכנים `GENDER` →
   מייצרים (`FROM`/`TO`) → בודקים ב‑`/voicetest.html`.
2. **קול לכל הכפתורים** — כרגע `fx-five/hug/kiss` ו‑`like-0..11` הם קליפ יחיד (Ayelet).
   ההורה רוצה שהם יישמעו **בקול של החבר שמבקרים אצלו**. צריך:
   - לייצר כל אחד **בכל קול בפלטה** (5 גרסאות) → ~75 קליפים.
   - שינוי אפליקציה: `voice.ts`/`FriendWorld` יבחרו את הווריאנט לפי הקול של החבר
     הנוכחי — כלומר **המגדר חייב לעבור ל‑`src/friends.ts`** + `voiceFor` בצד האפליקציה.
3. **פתיחת בובי (`intro-2`)** — "וואו!" ב‑Doron לא מושלם. **לנסות "wow" באותיות לטיניות**;
   אם לא נתפס — לשנות ל"יש! מצאתם אותי".
4. **אנגלית** — צריך נתוני‑אנגלית במתכון (תבניות, שמות, מילות‑מספר). קול Narakeet אנגלי.
5. **ניקוד מונע‑גמגום** ל‑INVITE עם "נשחק" (`בואו נשחק מחבואים/בכדור`) — לנקד `נְשַׂחֵק`.
