# 🎙️ קול טבעי לחברים (במקום הקול הרובוטי)

המשפטים הקבועים (היכרות כל חבר, ספירה, "כיף/חיבוק/נשיקה") מתנגנים מקובץ-קול **טבעי** מוקלט-מראש. אם הקובץ חסר — האפליקציה מקריאה בקול-הדפדפן (אז הכול עובד גם בלי זה).

---

## ✅ נפתר — קול נוירוני טבעי, חינם, בלי מפתח (Edge) · עודכן 03/06/2026

עברנו ל**קולות Microsoft Neural** דרך שירות ה-Read-Aloud של Edge — **חינם, בלי מפתח, בלי כרטיס**, ואיכות זהה ל-Azure. עברית: `he-IL-HilaNeural` (טבעי, לא רובוטי). הכל מיוצר אוטומטית (לא אחד-אחד) דרך `scripts/gen-voice.mjs` (ספק ברירת-מחדל `edge`, תלות `msedge-tts`).

- **עברית:** `node scripts/gen-voice.mjs` → `public/voice/he/` (103 קליפים: num-1..50, intro-0..49, fx). ✅ בוצע.
- **אנגלית (בהמשך, עם התרגום):** `$env:VOICE_LANG="en"; node scripts/gen-voice.mjs` (קול `en-US-AvaNeural`) → `public/voice/en/`. צריך קודם נתוני-אנגלית בסקריפט.
- **קול אחר:** `$env:EDGE_VOICE="he-IL-AvriNeural"` (גבר) וכו'.

> השאר במסמך זה (Narakeet / Google / ElevenLabs) נשמר כגיבוי — אבל **edge הוא הדרך**.

## 📍 (היסטוריה) סטטוס קודם · עודכן 02/06/2026

**מה כבר עובד:** לכל חבר יש "עולם" ב"⭐ החברים שלי" (לחיצה על חבר → היכרות מדוברת + ✋כיף/🤗חיבוק/😘נשיקה/🔢ספירה/🔊שוב). הקול כרגע = **קול-הדפדפן (TTS)** כי עוד לא ייצרנו קליפים. שכבת-הנגינה (`src/voice.ts` → `playClip`) תנגן אוטומטית קבצי `public/voice/<id>.mp3` ברגע שיהיו, אחרת נופלת ל-TTS.

**מה החלטנו — להשתמש ב-Narakeet:** עברית טובה במבטא ישראלי, 98 קולות (נשים: Ayelet / Tamar / Nurit / Hadas / Yael), **$6 חד-פעמי = 30 דק׳, לא פג ולא מנוי** — מספיק בענק (אנחנו צריכים ~2–3 דק׳). *(ElevenLabs = עברית חלשה. Gemini חינמי = רק 10 בקשות/יום. Google Cloud = עברית טובה אבל דורש להפעיל חיוב/כרטיס — נתקענו ביצירת המפתח.)*

**איך שולטים בקול ב-Narakeet (חקרנו):**
- **ניקוד מתקן הגייה** — `הַמִּסְפָּר` נשמע "מִספָּר" (number), בלי ניקוד `מספר` נשמע "מסַפֵּר" (מי שמספר סיפור).
- **נלהבות:** לעטוף ב-`**...**` + `!` + `(voice-speed: 1.1)` (לא slow!). אופציונלי שורה `(voice-emphasis: cheerful)` — תלוי קול.
- **לא לערבב** SSML `<speak>` עם טקסט רגיל (זאת הסיבה שזה "לא השתלב" בדמו).

**פרומפט-בדיקה (להדביק בדמו של Narakeet, לנסות קול-אישה):**
```
(voice-speed: 1.1)
**שָׁלוֹם! אֲנִי מִימִי, אֲנִי הַמִּסְפָּר חָמֵשׁ! הַצֶּבַע שֶׁלִּי טוּרְקִיז. הֲכִי כֵּיף לִי לִרְקוֹד. בּוֹאוּ נְשַׂחֵק!**
```

**מה נשאר (בסופ״ש):**
1. בדמו של Narakeet — לבחור **קול-אישה + סגנון** שאהבת (לוודא שמבטא "מספר" נכון ושנשמע נלהב).
2. להגיד לי איזה קול → אני: **אוסיף ספק `narakeet` לסקריפט**, **אנקד את כל הטקסטים** (מִסְפָּר, נְשַׂחֵק…), ואוסיף הדגשה/מהירות.
3. לקנות חבילת $6 + להשיג **מפתח-API** (בהגדרות Narakeet) → להריץ `node scripts/gen-voice.mjs`.
4. לבקש ממני **"גרסה"** → אעלה את הקליפים ו-Netlify יפרוס.

## ✅ מומלץ — Google Cloud TTS (העברית הכי טובה)

קולות עברית טבעיים. שכבה חינמית נדיבה (מיליון תווים בחודש) — ייתכן שתתבקשי **להפעיל חיוב** על הפרויקט, אבל **לא תחויבי** בתוך המכסה החינמית (השימוש שלנו זעיר).

1. https://console.cloud.google.com → פרויקט → להפעיל **Cloud Text-to-Speech API** → **Credentials → Create API key**.
2. מריצים (קול אישה, מואט):
   ```powershell
   $env:VOICE_PROVIDER="google"; $env:GOOGLE_TTS_KEY="המפתח"; node scripts/gen-voice.mjs
   ```
3. ```powershell
   git add public/voice; git commit -m "voice clips"; git push
   ```

- **קול:** ברירת המחדל `he-IL-Wavenet-C` (אישה). קולות עברית: `he-IL-Wavenet-A` ו-`C` = נשים, `B`/`D` = גברים.
  לבחירה אחרת: `$env:GOOGLE_TTS_VOICE="he-IL-Wavenet-A"`.
- **מהירות:** ברירת המחדל 0.85 (איטי יותר). להאט עוד: `$env:VOICE_SPEED="0.8"`.

## לשמוע קולות-אישה (ElevenLabs בלבד)
```powershell
$env:ELEVENLABS_API_KEY="KEY"; node scripts/gen-voice.mjs samples
