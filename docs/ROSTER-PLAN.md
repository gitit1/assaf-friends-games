## 🔧 מה צריך מימוש בקוד (כשמאשרים אצווה)

לכל חבר חדש, ההפעלה דורשת:
1. **`src/friends.ts`** — להוסיף לחבר `gender`, `special` (אינדקס פעילות), `game` (מזהה המשחק).
2. **`src/components/FriendWorld.tsx`** — להוסיף ערך ל‑`LIKES` עבור פעילות חדשה (אמוג'י + תווית + verb).
3. **`src/i18n/he.ts` + `en.ts`** — מפתח `world.like.<n>` לכל פעילות חדשה (מפתחות חייבים להתאים בין הקבצים).
4. **`scripts/gen-voice.mjs`** — `GENDER`, `INTRO_LIKE`, `SPECIAL` (3 משפטים), `FACTS` (4 עובדות) לכל חבר; ואז הקלטה בקול שלו.
5. **לעדכן** [GAMES.md](GAMES.md) — להעביר את המשחק מ"יתומים" ל"מקושרים" ולעדכן ספירה.
