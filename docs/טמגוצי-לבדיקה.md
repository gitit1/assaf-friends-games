# טמגוצ'י — מה לבדוק (QA) 🐣

רשימה לעבור עליה בנחת. פותחים את המשחק "החבר שלי" (הכרטיס עם 🐣).
לכל דבר: ✅ טוב / ✏️ לכוונן / ❌ לא עובד — ותכתבי לי.

## בחירת דמות
- [ ] לשונית **חבר** ↔ **חיה** עובדת, מדפדפים בחיצים
- [ ] 4 חיות מזוהות: **כלב 🐶, חתול 🐱, ארנב 🐰, אוגר 🐹**
- [ ] הדמות/חיה **עומדת על השטיח** (לא מרחפת) — גם במסך רחב

## אוכל 🍎 (כל הרצף — "סרטון")
- [ ] צועד **למטבח** (העולם נגלל, רגליים זזות)
- [ ] המקרר **מופיע ונפתח רק כשהגיע** (לא זז עם החדר)
- [ ] בוחרים אוכל → **מחזיק אותו ביד** כל הדרך
- [ ] **חוצה לחדר אוכל** (חלון/תמונה/שטיח — לא המטבח)
- [ ] **מניח על השולחן** ויושב לאכול (האוכל קטֵן נגיסה‑נגיסה)
- [ ] בסוף **חוזר לסלון**

## שתייה 🍼 / מים
- [ ] הולך, לוקח מיכל, ושותה

## אמבטיה 🧽 / שירותים 🚽
- [ ] **הולך לאמבטיה** (לא מיידי)
- [ ] **המים יורדים רק כשהגיע** למקלחת
- [ ] שירותים — יושב על האסלה

## שינה 😴
- [ ] חוצה ל**חדר שינה** אמיתי (קיר לילה + ירח)
- [ ] המיטה מופיעה כשהגיע, **נשכב** ומתכסה, Zzz, אור מתעמעם

## טיול 🚶
- [ ] יוצא דרך **דלת**
- [ ] עובר **כמה סצנות** (פארק → אגם עם ציפור → אחו עם פרפר)

## לכלוך + הלבשה
- [ ] כשמתלכלך — **הכתמים על הגוף**, וזזים איתו כשהולך
- [ ] **הלבשה** (לחבר): בגדים נראים **לבושים** (לא מונחים), חפצים **מוחזקים ביד**
- [ ] לחיה **אין** כפתור הלבשה
- [ ] לפני אמבטיה/שינה — **מוריד בגדים**

## כללי
- [ ] משחק (כדור) — החיה/חבר מופיע ומשחק
- [ ] חיבוק 🤗, ליטוף (לחיצה על הדמות)
- [ ] לבדוק גם בעברית וגם באנגלית
