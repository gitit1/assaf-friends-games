# טמגוצ'י — מה לבדוק (QA) 🐣

רשימה לעבור עליה בנחת. פותחים את המשחק "החבר שלי" (הכרטיס עם 🐣).
לכל דבר: ✅ טוב / ✏️ לכוונן / ❌ לא עובד — ותכתבי לי.

## בחירת דמות
- [ ] לשונית **חבר** ↔ **חיה** עובדת, מדפדפים בחיצים
- [ ] 4 חיות מזוהות: **כלב 🐶, חתול 🐱, ארנב 🐰, תוכי 🦜**
- [ ] הדמות/חיה **עומדת על השטיח** (לא מרחפת) — גם במסך רחב

## אוכל 🍎 (כל הרצף — "סרטון")
- [ ] צועד **למטבח** (העולם נגלל, רגליים זזות)
- [ ] המקרר **מופיע ונפתח רק כשהגיע** (לא זז עם החדר)
- [ ] בוחרים אוכל → **מחזיק אותו ביד** כל הדרך
- [ ] **חוצה לחדר אוכל** (חלון/תמונה/שטיח — לא המטבח)
- [ ] **מניח על השולחן** ויושב לאכול (האוכל קטֵן נגיסה‑נגיסה)
- [ ] בסוף **חוזר לסלון**

## שתייה 🍼 / מים
- [ ] הולך, לוקח מיכל, ושותה

## אמבטיה 🧽 / שירותים 🚽
- [ ] **הולך לאמבטיה** (לא מיידי)
- [ ] **המים יורדים רק כשהגיע** למקלחת
- [ ] שירותים — יושב על האסלה

## שינה 😴
- [ ] חוצה ל**חדר שינה** אמיתי (קיר לילה + ירח)
- [ ] המיטה מופיעה כשהגיע, **נשכב** ומתכסה, Zzz, אור מתעמעם

## טיול 🚶
- [ ] יוצא דרך **דלת**
- [ ] עובר **כמה סצנות** (פארק → אגם עם ציפור → אחו עם פרפר)

## לכלוך + הלבשה
- [ ] כשמתלכלך — **הכתמים על הגוף**, וזזים איתו כשהולך
- [ ] **הלבשה** (לחבר): בגדים נראים **לבושים** (לא מונחים), חפצים **מוחזקים ביד**
- [ ] לחיה **אין** כפתור הלבשה
- [ ] לפני אמבטיה/שינה — **מוריד בגדים**

## כללי
- [ ] משחק (כדור) — החיה/חבר מופיע ומשחק
- [ ] חיבוק 🤗 — הדמות **באמת מחבקת** (לא רק אמוג'י); ליטוף (לחיצה על הדמות)
- [ ] לבדוק גם בעברית וגם באנגלית

## בקיעה + שם + גדילה (חדש)
- [ ] חבר/חיה חדשים מגיעים כ**ביצה** — מחממים בלחיצות עד שבוקעת → ואז **נותנים שם**
- [ ] מתחילים כ**גור** (קטן, אוזניים קטנות, גוף שמנמן) → גדלים ל**מתבגר → בוגר** עם הטיפול
- [ ] החיות: כלב/חתול/ארנב **פלומתיים**, תוכי חלק (נוצות); כולם חמודים בכל שלב
- [ ] מוצגים שם + תג שלב (🐣 תינוק / 🌟 ילד / 👑 בוגר)

## מוד טיפול (הגדרות ⚙️)
- [ ] מתג **עדין** (ברירת מחדל) — החבר תמיד שמח, לא נחלש, בלי פרצוף עצוב/אפור
- [ ] מתג **רגיל** — הסטטים יורדים עם הזמן + מצב עצוב

## מספרים (אסף אוהב!)
- [ ] מונים שרק עולים: **ימים יחד** + **טיפולים**

## מעברים — ליטושים אחרונים
- [ ] הדלת **לא זזה** עם הדמות; ביציאה רואים **בית** בחוץ; חוזרים **דרך הדלת**
- [ ] הטיול = **סצנה רנדומלית אחת** (פארק / אגם / אחו)
- [ ] אחרי כל פעולה (אוכל/מקלחת/שינה/שתייה/שירותים) — **הולך חזרה** לסלון, לא קופץ
- [ ] שתייה בחדר אוכל; מקרר/שולחן/מיטה/אסלה הם רהיטים **בתוך** החדר

## אזור בדיקות 🧪 (כפתור בבית)
- [ ] הכפתור מוביל ל**אזור בדיקות**, ושם הקישורים עובדים:
  - [ ] בדיקת חיות (4 חיות × 3 שלבים)
  - [ ] בדיקת גלריה (תלת מימד)
  - [ ] בדיקת קול (שמות / מספרים / מילים)
