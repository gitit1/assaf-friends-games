# 🔍 מעבר מסודר על האפליקציה — חלק-חלק

עוברים יחד על כל חלק באפליקציה, לפי סדר והיגיון. לכל פריט 5 מימדים:

- **ערך** — מה הוא מלמד/נותן, והאם הוא שווה לאסף.
- **שיפור** — UX/מכניקה/ליטוש.
- **עברית** — תקין, טבעי, מנוקד היכן שמדובר (ולהוסיף קול נכון אם צריך).
- **אנגלית** — כל הטקסטים דרך מערכת ה-i18n (he/en).
- **אנימציה** — תנועה משמעותית ("juice"), לפי תכנית האנימציות.
- **רספונסיביות** 📱 — נראה ועובד מצוין בכל מסך (טלפון/טאבלט/מחשב). **חשוב מאוד** — נבדק בכל פריט.

> 🧊 **התלת-מימד (גלריה):** dev-only — הכפתור מוסתר ו-Three.js **כלל לא נטען** בפרודקשן (לא מבזבז משאבים).
> 📄 בסוף המעבר — לעדכן את ה-README.

## סדר העבודה (מוצע)

1. **מסגרת** (בית → קטגוריות → החברים שלי → עולם-חבר → הגדרות) — הבסיס שהכול יושב בו.
2. **מספרים** (הליבה לאסף) — משחק-משחק.
3. **חשיבה** → **כיף** → **יצירה** → **אותיות** → **ספורט**.

לכל פריט: אני מציגה ערך + רעיונות-שיפור, מחליטים יחד, ואז מבצעים (שיפור + עברית + אנגלית + אנימציה).

> אנגלית: רוב המשחקים עדיין עם עברית מוקשחת בקוד; ההמרה ל-i18n היא חלק מהמעבר. `[x]` = כבר הומר.
